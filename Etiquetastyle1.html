Imagen de fondo

Una imagen de fondo para un elemento div con la etiqueta style en la seccion head:

Puede especificar imágenes de fondo
para cualquier elemento HTML visible.
En este ejemplo, la imagen de fondo
se especifica para un elemento div.
De forma predeterminada, la imagen de fondo
se repetirá en la(s) dirección(es)
donde es más pequeño que el elemento
donde se especifica. (Intente cambiar el tamaño de la
ventana del navegador para ver cómo el
imagen de fondo se comporta.
