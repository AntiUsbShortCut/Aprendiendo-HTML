El elemento picture

Cambie el tamaño del navegador para ver diferentes versiones de la imagen cargando en diferentes tamaños de ventana gráfica. El explorador busca el primer elemento de origen donde la consulta de medios coincide con el ancho actual de la ventana gráfica del usuario, y recupera la imagen especificada en el atributo srcset.

El elemento img es necesario como la última etiqueta secundaria del bloque de declaración de imagen. El elemento img se utiliza para proporcionar compatibilidad con versiones anteriores para los navegadores que no admiten el elemento picture, o si ninguna de las etiquetas de origen coincide.

Nota: El elemento picture no es compatible con IE12 y versiones anteriores o Safari 9.0 y versiones anteriores.

El elemento picture

El elemento picture se puede utilizar cuando el formato de imagen no es compatible con todos los dispositivos.

El dispositivo utilizará el primer formato de imagen que admita, e ignorará el resto de las imágenes.
