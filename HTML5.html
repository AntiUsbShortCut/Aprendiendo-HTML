El atributo start

De forma predeterminada, una lista ordenada comenzará a contar a partir de 1. Utilice el atributo start para empezar a contar a partir de un número especificado:

  1. Café
  2. Té
  3. Leche
  1. Café
  2. Té
  3. Leche

Una lista anidada

Las listas se pueden anidar (lista dentro de la lista):

  1. Café
  2. Té
    1. Té negro
    2. Té verde
  3. Leche
    1. Leche descremada
    2. Leche sin lactosa
