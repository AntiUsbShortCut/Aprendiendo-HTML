El elemento base

- Tenga en cuenta que sólo hemos especificado una dirección relativa para la imagen. Puesto que hemos especificado una URL base en la sección head, el navegador buscará la imagen en"https://www.w3schools.com/images/stickman.gif".

Etiqueta base HTML - Tenga en cuenta que el enlace se abre en una nueva ventana, incluso si no tiene ningún atributo de destino "_blank". Esto se debe a que el atributo de destino del elemento base se establece en "_blank".
