El elemento de código

Ejemplo de código de programación:

x = 5; y = 6; z = x + y;

El elemento de code no conserva el espacio en blanco ni los saltos de línea.

Para solucionar este problema, puede colocar el elemento de code dentro de un elemento pre:

  
x = 5;
y = 6;
z = x + y;
