TIPOS DE ENTRADAS HTML

Este capítulo describe los diferentes tipos de elementos HTML input.

1.-Tipo de entrada text

El input type="text" define un campo de entrada de texto de una sola línea






1.- Tipo de entrada password

El input type="password" define un campo de contraseña






2.- Tipo de entrada submit

El input type="submit" define un botón para enviar datos de formulario a un administrador de formularios.
El gestor de formularios suele ser una página de servidor con un script para procesar datos de entrada.
El controlador de formulario se especifica en el atributo action del formulario

3.- Tipo de entrada reset

El input type="reset" define un botón de reinicio que reiniciará todos los valores del formulario a sus valores predeterminados






4.- Tipo de entrada radio

El input type="radio"define un botón de radio.
Los botones de opción permiten al usuario seleccionar SÓLO UNA de un número limitado de opciones





5.-Tipo de entrada checkbox

El input type="checkbox" define una casilla de verificación.
Las casillas de verificación permiten al usuario seleccionar CERO o MÁS opciones de un número limitado de opciones.

6.- Tipo de entrada button

El input type="button" define un botón

7.- Tipo de entrada color

El input type="color"utiliza para campos de entrada que deben contener un color.
Según la compatibilidad del navegador, puede aparecer un selector de color en el campo de entrada.

8.- Tipo de entrada date

El input type="date" utiliza para campos de entrada que deben contener una fecha.
Dependiendo de la compatibilidad del navegador, puede aparecer un selector de fecha en el campo de entrada.

También puede usar los atributos miny max para agregar restricciones a las fechas





9.- Tipo de entrada datetime-local

El input type="datetime-local" especifica un campo de entrada de fecha y hora, sin zona horaria.
Dependiendo de la compatibilidad del navegador, puede aparecer un selector de fecha en el campo de entrada.

10.- Tipo de entrada email

El input type="email" utilizada para campos de entrada que deben contener una dirección de correo electrónico.
Según la compatibilidad del navegador, la dirección de correo electrónico se puede validar automáticamente cuando se envía.
Algunos teléfonos inteligentes reconocen el tipo de correo electrónico y agregan ".com" al teclado para que coincida con la entrada del correo electrónico.

11.- Tipo de entrada file

El input type="file" define un campo de selección de archivo y un botón "Examinar" para la carga de archivos.



12.- Tipo de entrada month

El input type="month" permite al usuario seleccionar un mes y el año.
Dependiendo de la compatibilidad del navegador, puede aparecer un selector de fecha en el campo de entrada.

13.- Tipo de entrada number

El input type="number" define un campo de entrada numérico.
También puede establecer restricciones sobre los números que se aceptan.
El siguiente ejemplo muestra un campo de entrada numérico, donde puede ingresar un valor de 1 a 5.

Pasos numéricos
Según la compatibilidad del navegador: se aplicarán pasos fijos en el campo de entrada.

14.- Tipo de entrada range

El input type="range" define un control para ingresar un número cuyo valor exacto no es importante (como un control deslizante). Rango por defecto es de 0 a 100. Sin embargo, se puede establecer restricciones en lo que los números son aceptados con los atributos min, max y step

15.- Tipo de entrada search

El input type="search" utiliza para campos de búsqueda (un campo de búsqueda se comporta como un campo de texto normal).

16.- Tipo de entrada tel

El input type="tel" utiliza para campos de entrada que deben contener un número de teléfono.





Format: 123-45-678

17.- Tipo de entrada time

El input type="time" permite al usuario seleccionar una hora (sin zona horaria).
Según la compatibilidad del navegador, puede aparecer un selector de tiempo en el campo de entrada.

18.- Tipo de entrada url

El input type="url" utiliza para campos de entrada que deben contener una dirección URL.
Dependiendo de la compatibilidad del navegador, el campo de URL se puede validar automáticamente cuando se envía.
Algunos teléfonos inteligentes reconocen el tipo de URL y agregan ".com" al teclado para que coincida con la entrada de URL.

19.- Tipo de entrada week

El input type="week" permite al usuario seleccionar una semana y un año.
Dependiendo de la compatibilidad del navegador, puede aparecer un selector de fecha en el campo de entrada.
